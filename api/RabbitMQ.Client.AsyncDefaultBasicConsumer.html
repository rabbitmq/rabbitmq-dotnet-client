Called each time a message is delivered for this consumer.

Remarks

This is a no-op implementation. It will not acknowledge deliveries via BasicAck(UInt64, Boolean) if consuming in automatic acknowledgement mode. Subclasses must copy or fully use delivery body before returning. Accessing the body at a later point is unsafe as its memory can be already released.
